About This Tool

This is a streamlined assistant tool designed to simplify everyday workflow in radiation therapy, such as "treatment calendar counting", "BED/EQD2 calculations", and "dose percentage slicing". All data processing is executed entirely within your browser (locally). If you download the HTML file, it runs completely offline without sending any data to external servers.

For Customization & Feature Additions

If you wish to add your own customized dose tiers or change the visual layout, the easiest way is to feed this file directly into generative AIs (such as ChatGPT or Gemini) and tell them what you want.

Download the HTML file using the button below, upload it into an AI chat, and simply prompt it with: "Add feature X to this code" or "Modify section Y". The AI will easily handle the coding part for you.
